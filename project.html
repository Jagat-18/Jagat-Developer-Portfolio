boAt-Clone Project

Overview:
The boAt Clone project is a modern e-commerce website UI built to replicate the visual and functional experience of the popular boAt website. It demonstrates responsive layout, intuitive product cards, category-based navigation, and polished UI components using HTML, CSS, JavaScript & Bootstrap.
